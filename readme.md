## Controls:

| Key    | Action   |
| ------ | -------- |
| RIGHT-ARROW | Volume + |
| LEFT-ARROW | Volume - |
| W | Accelerate |
| A | Rotate Counter-Clockwise |
| S | Rotate Clockwise |
| SPACE | Shoot |
